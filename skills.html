أمثلة محلولة

المثال الأول: x² - 5x + 6 = 0

الحل: x = 2 أو x = 3

التحليل: (x - 2)(x - 3) = 0

المثال الثاني: 2x² + 4x - 6 = 0

الحل: x = 1 أو x = -3

التحليل: 2(x - 1)(x + 3) = 0

المثال الثالث: x² + 4x + 5 = 0

الحل: لا يوجد حلول حقيقية (المميز سالب)

الحلول المركبة: x = -2 ± i
