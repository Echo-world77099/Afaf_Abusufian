Solved Examples

Example 1: x² - 5x + 6 = 0

Solution: x = 2 or x = 3

Factorization: (x - 2)(x - 3) = 0

Example 2: 2x² + 4x - 6 = 0

Solution: x = 1 or x = -3

Factorization: 2(x - 1)(x + 3) = 0

Example 3: x² + 4x + 5 = 0

Solution: No real solutions (negative discriminant)

Complex solutions: x = -2 ± i
